# Variable Declaration

- JavaScript Variables can be declared in 4 ways:

    - Automatically
    - Using var
    - Using let
    - Using const

 ## Automatic

 - They are automatically declared when first used:

```javascript
v = 12;
```

## Var keyword

```javascript
var v = 12
```

## Let keyword

```javascript
let v = 12
```

## Const keyword

```javascript
const v = 12
```
